Dit is een handler waarmee je kunt communiceren tussen verschillende banken
dit is een prototype dat gestopt moest worden omdat de structuur hiervan ervoor
zorgt dat het alleen lokaal werkt en ik geen toegang heb tot de website waarin ik
kan communiceren als ik het verbind met de bank. Het is wel mogelijk json objecten
te versturen en te ontvangen tussen verschillende banken en die worden gecontorleerd
op de inhoud.
